## Getting Started

Follow these steps to get started with Model Playground:

### Prerequisites

- Node.js (v16+)
- pnpm

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Yat3s/model-playground.git
   cd model-playground
   ```

2. Install dependencies:

```bash
pnpm install
```

3. Set up environment variables by creating a .env file:

```env
API_KEY=your_api_key_here
DATABASE_URL=your_database_url_here
```

4. Start the development server:

```
pnpm dev
```

5. Open your browser and navigate to
```
http://localhost:3000
```

## Configuration Management System (CMS)

### Features:
- **Drag-and-Drop Interface**: Quickly upload new models.
- **Tagging and Categorization**: Organize models for quick access.
- **Version Control**: Track and roll back changes to model configurations.

### Accessing the CMS:
1. Navigate to the `/cms` endpoint in your browser.
2. Log in using your admin credentials.
3. Start managing your models efficiently.
